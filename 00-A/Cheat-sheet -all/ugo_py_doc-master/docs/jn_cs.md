# Function

docstrings, info, source.

```python
def print_number(nb):
    
    """
    The docstring. print_number prints Hi!
    """
    print("Hi!" * nb)
```

```python
print_number(1)
print_number(5)
```

    Hi!
    Hi!Hi!Hi!Hi!Hi!

```python
pdef print_number
```

     print_number(nb)
     
Open a window about the docstrings.

```python
pdoc print_number

```

Open a window about the function.

```python
pinfo print_number

```
